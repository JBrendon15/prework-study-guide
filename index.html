HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements that are nested inside another element are called child elements.
  • The element that holds the nested elements is called the parent element

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Declarations include two important components: the CSS property and the value of the property.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new brach and switches to it
  • git checkout branch-name: switches to the branch
  • git clone (SSH Value): clones a GitHub repository onto your computer
  • git add: adds modifications in the current working branch to the staging area
    • -A: flag that indicates we want to add all changes
  • git commit: creates a commit
    • -m: flag that gives a description of what we are committing
  • git pull: receive a branch's modifications into the local environment (computer)
    • origin: keyword that indicates the source of the pull will be in the GitHub repo
    • main: keyword that indicates main branch
  • git push: pushes the changes we have done locally to our remote GitHub branch
